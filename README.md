# **SpeX 🌍 – AI-Powered Smart Glasses for Visually Impaired**

SpeX is an **AI-powered smart glasses solution** designed to help visually impaired individuals navigate the world independently. With real-time visual assistance, auditory feedback, and proactive alerts, SpeX enhances mobility, safety, and social interaction.

---

## **💡 Problem Statement**
Visually impaired individuals face challenges in navigation, interaction, and accessing information. Traditional tools lack real-time AI-driven assistance.

SpeX integrates:
- Object detection
- Face recognition
- Text-to-speech
- Geographical tracking
- Distance measurement
- Alerts  
All within wearable smart glasses.

---

## **✨ Key Features**
- **🔄 Object Detection (YOLO):** Identifies obstacles and points of interest.  
- **👤 Face Recognition:** Enhances social interactions.  
- **📃 Text Recognition (OCR):** Reads printed text aloud.  
- **🌐 Geographical Tracking:** Real-time location awareness.  
- **🔭 Distance Measurement:** Detects obstacles safely.  
- **⚠ Alerts:** Proactively warns of hazards.  
- **🎤 Voice Interaction:** Enables hands-free operation.

---

## **🔧 Installation & Setup**
1. Install **Python 3.11**.  
2. Clone the repository:  
   ```bash
   git clone https://github.com/harshendram/speXweb.git
   ```  
3. Navigate to the directory:  
   ```bash
   cd speXweb
   ```  
4. Install dependencies:  
   ```bash
   pip install -r requirements.txt
   ```  
5. Run the app:  
   ```bash
   python speX.py
   ```

---

## **🔊 Usage**
- **"What is the distance to [object]?"** – Measures distances.  
- **"Identify objects around me."** – Detects nearby objects.  
- **"Who is that?"** – Recognizes faces.  
- **"Read this text."** – Reads visible text.  
- **"Where am I?"** – Provides real-time location.
